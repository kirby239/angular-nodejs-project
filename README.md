## API Endpoints

### Login

**URL:** `POST http://localhost:3000/auth/login`

**Body:**
```json
{
    "username": "superadmin",
    "password": "superadmin"
}
```

### Register

**URL:** `POST http://localhost:3000/auth/register`

**Body:**
```json
{
    "username": "example",
    "password": "example"
    "email":"example@gmail.com",
    "name":"exampl"
}
```
